# Validador de CPF
Algoritmo criado em python puro sem usar nenhuma biblioteca
para verificar se o cpf é válido ou não.

## Foco do programa
- Solicitar um CPF ao usuário (com ou sem pontuação)

- Verificar se o CPF é válido de acordo com o algoritmo oficial

- Informar se o CPF é válido ou inválido

- Permitir validar vários CPFs até o usuário decidir sair
