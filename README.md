# Python Exercises

Repositório criado para **praticar lógica, algoritmos e boas práticas em Python**, com projetos pequenos e progressivos, simulando situações próximas do dia a dia de um desenvolvedor.

O foco principal é **aprender fazendo**, com código organizado, testes automatizados e separação clara de responsabilidades.

---

## Estrutura do repositório



python-exercises/
│
├── 01_calculadora_cli/
│ └── Calculadora em linha de comando com validações básicas
│
├── 02_validador_cpf/
│ └── Validação de CPF com implementação manual e testes
│

## Projetos

### 01 - Calculadora CLI
Projeto simples em linha de comando para praticar:
- Estruturas condicionais
- Laços de repetição
- Tratamento de erros
- Organização básica de funções

---

### 02 - Validador de CPF
Projeto focado em lógica e testes:
- Implementação manual do algoritmo de validação de CPF
- Separação entre lógica e execução
- Testes automatizados com `pytest`
- Escrita de código legível e testável

---

### 03 - Controle de Usuários (Mini CRUD)
Mini sistema de gerenciamento de usuários via terminal.

Funcionalidades:
- Criar usuários
- Listar usuários
- Buscar usuários
- Atualizar usuários
- Persistência em arquivo (JSON)
- Testes automatizados

Objetivos de aprendizado:
- Organização de código por responsabilidade
- Funções puras e testáveis
- Uso de testes como proteção
- Simulação de um CRUD simples sem frameworks

> Este projeto foi intencionalmente mantido simples, adequado ao nível atual, priorizando clareza e aprendizado ao invés de complexidade desnecessária.

---

## Tecnologias utilizadas

- Python 3
- Pytest
- JSON para persistência simples de dados
- Git e GitHub para versionamento

## Como executar os projetos

1. Clone o repositório:
```bash

Entre na pasta do projeto desejado:

cd python-exercises/03_controle_usuario
