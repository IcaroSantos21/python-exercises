# Calculador CLI

Programa criado para trabalhar como uma calculadora, fazendo operações e imprimindo o resultado.

## Linguagem:
- Python 3.14.2

## Conceitos
- Estrutura *while*
- Estrutura *if* / *else* / *elif*
- Conversão de tipos
- Tratamento de erros
- Entrada de dados
- Funções em Python
- Boas práticas para funções pequenas e bem definidas
